# Verilog & Digital Electronics Practice Lab 

Welcome! This repository is my personal sandbox where I document my daily journey, experiments, and logic designs using Verilog HDL. 

Instead of just following a fixed textbook, I use this space to build, simulate, and verify digital circuits from scratch—ranging from basic logic gates to complex finite state machines.

## Current Progress & Roadmap

### 🔹 Combinational Logic
- [x] **Adders:** Half Adder & Full Adder (`Gate-level` & `Dataflow`)
- [x] **Subtractors:** Half Subtractor & Full Subtractor
- [x] **Decoders:** 2-to-4 Line Decoder (with Enable logic)
- [ ] **Multiplexers:** 2:1, 4:1 MUX & DEMUX
- [ ] **Code Converters:** Binary to Gray / Gray to Binary converters

### Sequential Logic & Storage (Up Next ⏳)
- [ ] **Flip-Flops:** SR, D, JK, and T Flip-Flops (with characteristic equations)
- [ ] **Registers & Counters:** Shift Registers (SISO, SIPO, PISO, PIPO) & Synchronous MOD Counters
- [ ] **Finite State Machines (FSM):** Mealy and Moore sequence detectors

##  Tech Stack & Workflow
- **HDL:** Verilog
- **Simulation & Waveforms:** Icarus Verilog (`iverilog`), VVP, and GTKWave (for `.vcd` files)
- **IDE:** Visual Studio Code / Xilinx Vivado
